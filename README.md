# GitPulse

A GitHub profile analytics dashboard built with React + Recharts.

🔗 **Live Demo:** https://gitpulse-indol.vercel.app/

## Features
- Profile card with avatar, bio, location
- Stats: repos, stars, forks, followers
- Top 5 repos sorted by stars
- Language distribution donut chart
- Most recent activity
- Skeleton loaders & error handling

## Tech Stack
- React + Vite
- Recharts
- GitHub Public REST API (no auth required)

## Run Locally
npm install
npm run dev
